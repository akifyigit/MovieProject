API Documentation

Base URL

The base URL for all endpoints is:

http://localhost:3000/api/media

Endpoints

Retrieve All Media Items

  • Endpoint: /media
  • HTTP Method: GET
  • Description: Retrieves all media items stored in the database.
  • Request Body: None
  • Response:
    • Status Code: 200 (OK)
    • Content-Type: application/json
    • Body: An array of media items in JSON format.

Example

    
      GET /api/media
      
      Response:
      [
        {
          "id": 1,
          "name": "Movie 1",
          "year": "2023",
          "genre": "Action",
          "poster": "http://example.com/poster.jpg",
          "description": "Description of Movie 1"
        },
        ...
      ]

Add a New Media Item

  • Endpoint: /media
  • HTTP Method: POST
  • Description: Adds a new media item to the database.
  • Request Body: JSON object with the following properties:
    • name (string): Name of the media item
    • year (string): Year of release
    • genre (string): Genre of the media item
    • poster (string): URL of the poster image
    • description (string): Description of the media item
  • Response:
    • Status Code: 201 (Created)
    • Content-Type: application/json
    • Body: JSON object with a success message

Example

  
    POST /api/media
    
    Request Body:
    {
      "name": "New Movie",
      "year": "2024",
      "genre": "Drama",
      "poster": "http://example.com/new_movie.jpg",
      "description": "Description of the new movie"
    }
    
    Response:
    {
      "message": "Media item added successfully"
    }

Update Media Item

  • Endpoint: /media/:id
  • HTTP Method: PUT
  • Description: Updates a specific media item in the database.
  • Request Body: JSON object with the properties to be updated:
    • name (string): Updated name of the media item
    • year (string): Updated year of release
    • genre (string): Updated genre of the media item
    • poster (string): Updated URL of the poster image
    • description (string): Updated description of the media item
  • Response:
    • Status Code: 200 (OK)
    • Content-Type: application/json
    • Body: JSON object with a success message

Example

      
        PUT /api/media/:id
        
        Request Body:
        {
          "name": "Updated Movie",
          "year": "2025",
          "genre": "Comedy",
          "poster": "http://example.com/updated_movie.jpg",
          "description": "Updated description of the movie"
        }
        
        Response:
        {
          "message": "Media item deleted successfully"
        }

Get Spesific Media Item

  • Endpoint: /media/:id
  • HTTP Method: GET
  • Description: Retrieves a specific media item from the database based on its ID.
  • Request Parameters:
    • id (integer): ID of the media item to retrieve
  • Response:
    • Status Code: 200 (OK)
    • Content-Type: application/json
    • Body: JSON object containing the media item with the specified ID

Example

  
    GET /api/media/123
    
    Response:
    {
      "id": 123,
      "name": "Example Movie",
      "year": "2023",
      "genre": "Action",
      "poster": "http://example.com/example_movie.jpg",
      "description": "Description of the example movie"
    }

Delete Media Item

  • Endpoint: /media/:id
  • HTTP Method: DELETE
  • Description: Deletes a specific media item from the database based on its ID.
  • Request Parameters:
    • id (integer): ID of the media item to delete
  • Response:
    • Status Code: 200 (OK)
    • Content-Type: application/json
    • Body: JSON object with a success message

Example

  
    DELETE /api/media/123
    
    Response:
    {
      "message": "Media item deleted successfully"
    }
